# Distributed Banking System Simulator — Python

This repository contains the **Python implementation** of a distributed banking system simulator developed as a final project for the **Operating Systems** course.

The project models a banking environment in which multiple branches and transactions operate concurrently, demonstrating important OS concepts such as **processes, threads, scheduling, synchronization, critical sections, and deadlock handling**.

## Features

- Simulation of distributed bank branches
- Multi-threaded transaction execution
- CPU scheduling mechanisms
- Starvation prevention using aging
- Critical section handling with mutex
- Deadlock detection and recovery
- Optional GUI-based simulation
